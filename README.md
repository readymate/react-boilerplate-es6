# Readymate Boilerplate ES6

[![Greenkeeper badge](https://badges.greenkeeper.io/readymate/react-boilerplate-es6.svg)](https://greenkeeper.io/)

```bash
# install dependencies
yarn

# start project
yarn start

# build project
yarn build

# deploy project (now)
yarn build

# format project
yarn format

# lint project
yarn lint

# test project
yarn test
```
